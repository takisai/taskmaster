      Task Masterについて

      Task Masterは、タスク管理ができるページです。

      製作者
      takisai
      Version 1.0
      とりあえず作った
      動作ブラウザ
      使い方

      Task Masterは、基本的に上から2行目の入力欄に入力することで操作します。以下の説明に出てくる文字は、全て半角文字です。

      タスクの設定
      タスク入力

      時刻設定文字列 [ / タスク設定 [ / タスク名 ] ]

      タスク名を省略した場合、時刻設定文字列がタスク名になります。

      例: "30" ←30分後にデフォルト設定でタスクを設定。タスク名は「30」
      "7:00/6/wake-up" ←朝7時に6番の音を鳴らすタスクを設定（重要度は既定値）。タスク名は「wake-up」

      時刻設定文字列

      時間指定 [ + ( 時刻設定文字列 | "" ) ]

      時間指定を+でつなげることで、設定した時刻に自動で次の時間指定を行うことができます。+で指定を終えると、直前の時間指定が繰り返されます。

      例: "10+5+5" ←10分後、それから5分後(=15分後)、さらに5分後(=20分後)に設定。
      "7:30+5+" ←朝7時30分、それから5分ごとに設定(いわゆるスヌーズ)。

      参考: +でつなげた時間指定は全て同じタスク設定/タスク名が適用されます。回避するためにはマクロを利用します。

      時間指定

      タイマー文字列 | アラーム文字列

      タイマー文字列

      [ D, ] HM [ .S ]

      Dは日を表す数値です。HMは下二桁が分を、残りの数字が時間を表す数値です。Sは秒を表す数値です。

      例: "130" ←1時間30分後を示します。
      "3,2400.15" ←3日と24時間と15秒後(=4日と15秒後)を示します。

      アラーム文字列

      [ [ Y- ] M-D, ] h : m [ :d ]

      Y、M、D、h、m、dはそれぞれ年、月、日、時、分、秒を表す数値です。

      それぞれの数値は省略することができます。省略した単位の場所によって挙動が異なります。省略した数値より上の単位に数値がある場合、月日なら1が、時刻なら0が入っているとみなします。そうでない場合、入力された数値と全て一致する次の時刻に設定します。

      例: "23:0" ← 23時0分0秒を表します。これを入力したのが23時00分より前の場合、その日の23時が指定されます。23時00分より後の場合は、次の日の23時ちょうどが指定されます。
      "1-,9:0" ← 1月1日9時0分0秒を表します。入力したのが1月1日の9時より前の場合、その年の1月1日9時が指定されます。そうでない場合は、次の年の1月1日9時が指定されます。

      タスク設定

      [ 音番号 ] [ 重要度 ]

      音番号は0から9の数値または-(ハイフン)。数値を入れると、対応する番号の音が鳴ります。-の場合は音が鳴りません。

      重要度は.(ピリオド)または!、!!、!!!のいずれかです。重要度の指定は、設定した時刻からの動作を変更します。.の場合、15秒後にリストから消去されます。!の場合、タスクが黄色で強調されて自動消去はされません。!!の場合、タスクは赤色で強調されて、さらにTask Master全体が赤っぽく表示されます。!!!の場合、!!に加えてアラートが表示されます。

      また、どちらの設定も省略することができます。その場合、既定値が適用されます。

      例: "!!" ←音は既定値、重要度は高でタスクを設定

      タスク名

      タスク名は;(セミコロン)以外の任意の文字列が利用できます。
