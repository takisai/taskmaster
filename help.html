Task Masterについて

1 概要

Task Masterは、決められた時刻に音を出すことができるWEBツールです。

1.1 諸情報

  • 最新版: Version 1.0 (2019-xx-xx)
  • 製作者: takisai
  • 動作環境: Chrome・Firefoxなら問題なく動くはずです。
    IE・Edgeは動かないと思います。スマホは未確認。

2 使い方

Task Masterは、基本的に上から2行目の入力欄にコマンドを入れてEnterキーを押すことにより動作します。

2.1 タスクの入力

タスクは、「(1)ある時間経過したら/ある時刻になったら、(2)どういう音で知らせるか/どのくらい重要なのか、(3)表示する名前」の3つを入力して設定することができます。

2.1.1 ある時間経過したら(タイマー)/ある時刻になったら(アラーム)

「ある時間経過したら(タイマー)」という設定は、数字を並べて表します。例えば、3分なら3と入力します。また、4分33秒のように秒単位で指定したいときは、4.33と<分>.<秒>のように表します。ただし、<分>は2桁までの数値で、3桁以上の場合は「右側2桁が<分>、残りが<時間>」を表します。例えば、1時間5分なら105のようになります。1日単位の指定もすることができて、それは一番左に<日>,と書きます。例えば、10,537.43は10日と5時間37分43秒を表しています。

「ある時刻になったら(アラーム)」という設定は、単位ごとに区切り文字を入れて表します。例えば、21時38分は21:38のようになります。7時38分46秒なら7:38:46のように表します。また、月日単位の指定もできます。例えば、3月7日の22時なら3-7,22:0のように書きます。年単位の表現は2019-5-12,16:10:00のように書きます。

アラームで設定される時間は、書かれた時刻と等しくなる一番始めの時間になります。例えば、朝8時に13:00を指定すると、5時間後にタスクが設定されます。しかし、15時に13:00と指定すると、次の日の13時にタスクが設定されます。

2.1.2 どういう音で知らせるか/どのくらい重要なのか

Task Masterは10種類の音を出すことができます。それぞれ0から9までの数字が割り当てられているので、それを指定すればOKです。音の確認はsound 0のようにコマンドを使ってください。

どれくらい重要なのかを指定すると、時間になったときの表示が変わります。

  • 低い重要度(.と表します)のとき
    しばらくすると勝手にタスクのリストから消えます。
  • 中くらいの重要度(!と表します)のとき
    時間になるとそのタスクが黄色で強調されます。勝手に消されることはありません。
  • 重要度が高い(!!と表します)とき
    時間になると赤色で強調され、Task Masterの画面全体も赤っぽくなります。
  • 重要度がさらに高い(!!!と表します)とき
    重要度が高いときの表示に加えて、ブラウザのアラートが機能します。(この動作を行うと他の処理が止まってしまうため、!!!の指定は推薦しません。)

これらの設定は、単純につなげるだけで表現することができます。例えば、1番の音を出してほしい&重要度を中に指定するときは、1!のように指定します。また、これらの設定は省略することができます。省略した設定は、既定値が適用されます。

2.1.3 表示する名前

タスク名は、;(セミコロン)以外の文字が使えます。省略すると、2.1.1で指定した文字列がタスク名になります。

2.1.4 タスクの入力まとめ

2.1.1・2.1.2・2.1.3を/でつなぐと、タスクの入力をすることができます。例えば、12時に5番の音を出す重要度低のタスク「昼休み」を設定するときは、12:00/5./昼休みと入力します。タスク名は省略することができて、そのときは12:00/5.のように/も省略します。また、音や重要度の設定も省略するときは、12:00のように最初の/も省略することができます。

2.1.5 繰り返しタスクの入力をする

Task Masterは、繰り返しタスクを設定する(いわゆるスヌーズ)こともできます。例えば、朝6時30分に重要度高で0番の音を鳴らす「GOOD MORNING!」というタスクは6:30/0!!/GOOD MORNING!ですが、これを6:30+5+/0!!/GOOD MORNING!と書き換えることで、さらに5分おきのスヌーズをしてくれます。

どのように書いて、どういう動作をするのか説明します。書き方は2.1.1で作る文字列を+でつなげるだけです。動作の例を出すために、10+2+3というタスクの指定を考えます。この文字列は、最初に10分のタイマーを設定します。10分経って音が鳴るときに、新たに2+3という設定がされます(これは2分のタイマーですね)。同様に2分が経過すると、音が鳴ると同時に3というタスクが設定されます。

また、+でつなげる文字列を+で終わらせると、最後の指定がずっと繰り返されるようになります。例えば5+というタスクなら、5分おきにアラームが繰り返されるようになります。

ただし、これで新たに追加されるタスクの音や重要度は、最初に指定したものと同じになります。最初の例(6:30+5+/0!!/GOOD MORNING!)なら、5分おきに重要度高&0番の音が鳴る「GOOD MORNING!」というタスクが繰り返し設定されるということです。

2.1.6 タスクで指定した時間に別のコマンドを実行したい

タスクで指定した時間に、別のコマンドを実行することができます。ただし、実行できるコマンドは;や/が入っていないこと、最後の文字が.や!でないことが必要です。例えば、5分後にhogeというコマンドを実行したい場合は5/*hogeのように<音番号>の部分を*<実行したいコマンド>に置き換えます。もちろん、重要度やタスク名を設定することもできます。

条件に合わないコマンドを実行するためには、2.7で説明するマクロを利用する必要があります。

2.2 タスクの入力(別の方法)

2.1ではタスクの入力方法を紹介しましたが、menu>タスク設定からでも設定することができます。

2.3 タスクの削除

タスクの削除は2つ方法があります。1つ目はタスクリストの左側にあるremoveボタンを押す方法です。

もうひとつは、removeコマンドを使う方法です。タスクリストの一番左の番号をremoveの後に書くことで、そのコマンドが消去されます。また、番号は複数書くことができます。例えば、remove 3 4 5なら3から5番目のタスクを消去します。また、一度にすべてのタスクを消去するにはremove *を指定します。

さらに、番号なしのremoveコマンドを使うと、設定した時間が来て取り消し線が引かれたタスクを全て消去することができます。

2.4 タスクの既定値の設定

2.1.2で書いた既定値は変更することができます。書式はdefault <音番号><重要度>です。<音番号><重要度>のどちらも、2.1.2と同じ方法で記述します。また、どちらかを省略することもできます。例えば、default !!は規定の音はそのまま、重要度だけ高に変更するというコマンドです。

また、現在設定されている既定値はdefaultで確認することができます。

2.5 タスクの表示方法の切り替え

初期設定だと、タスクは名前の後ろに(18:30:00)のような文字列が表示されています。この例は「18時30分にタスクは音を鳴らします」という意味です。これを「あとどのくらいで音が鳴るのか」に変更することができます。コマンドswitchを使うと、「いつ鳴らすか」と「あとどれだけで鳴るのか」の表示を切り替えることができます。

2.6 音を止める/音量操作

Task Masterの音を止める方法は2つあります。1つはタスクの右側にあるstopボタン(音が鳴っているときしか現れません)を押す方法です。

もうひとつはstopコマンドによる方法です。removeコマンドと同様に、タスクリスト一番左の番号を使って、そのタスクの音を止めることができます。また、stop *とすることで、すべての音を止めることができます。

音量はvolumeバーで操作するか、volume 50のようにコマンドで指定します。音量を示す数値は0から100で、数値が大きいほど音が大きくなります。

2.7 マクロ

マクロを使うと、例えばramenと入力すれば、自動で3//ramenというコマンドに置き換えることができます。

マクロの指定方法は<元文字列>-><置き換え後文字列>です。ただし、<元文字列>に->を含めることはできません。<元文字列>と<置き換え後文字列>はJavaScriptのreplaceメソッドを利用しているため、複雑な指定をすることができます。例えば、dup (.*)$->$1;button $1というマクロを指定しておくと、dup 50という文字列は50;button 50というコマンドに置き換えられます。詳しくは、JavaScriptのreplaceメソッドを参照してください。(参考: JavaScript replace - Google 検索)

マクロによる置き換えは、登録順にすべてのマクロで行われます。例えば、a->bとb->cというマクロをこの順番に登録した場合、aというコマンドはcに置き換えられます。マクロでの置換が行われないようにするには、先頭に#をつけてコマンドを入力します。

マクロの一覧はshow-macroコマンドで見ることができます。マクロの消去は、マクロリストの左にあるremoveボタンを押すか、remove-macroコマンドにより行います。removeコマンドと同様に、マクロの番号を指定することで消去できます。マクロ一覧を隠すときはhide-macroを実行します。

2.8 ボタンの作成

今まで書いてきたコマンドは、ボタンを押すことで実行することもできます。buttonコマンドの後ろに入力したいコマンドを入れて実行すると、volume操作バーとタスクリストの間にボタンが現れます。このボタンを押すことで、登録された内容が実行されます。例えば、button switchというコマンドで作成したボタンは押すたびにswitchコマンドを実行します。

ただし、コマンドに;を含めることはできません。

2.9 一度に複数のコマンドを実行する

マクロを除くこれまでのコマンドは、;(セミコロン)でつなぐことで一度にすべて実行することができます。
